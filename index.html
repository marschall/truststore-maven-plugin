truststore Maven Plugin

The truststore Maven Plugin creates a PKCS12 truststore from certificate files in a project.

When no configuration values are set, the plugin sets default values.

Goals Overview

The truststore Plugin has only one goal:

  • truststore:pkcs12 creates a PKCS12 truststore from the certificates in the source folder.

Usage

General instructions on how to use the truststore Maven Plugin can be found on the usage page.

An example on how to use the truststore Maven Plugin can be found in the truststore-maven-plugin-examples repository.
